Rashun

A macOS menu bar app that tracks your AI coding tool usage across multiple sources — so you always know where you stand.

Install with one command
curl -fsSL https://raw.githubusercontent.com/alexanderheffernan/rashun/main/install.sh | bash
macOS 14+ Swift 6.2 MIT License Open Source
See it in action

Designed to stay out of your way

Rashun lives in your menu bar and gives you an at-a-glance view of your remaining quota across every AI tool you use.

Features

Everything you need, nothing you don't

Built for developers who juggle multiple AI coding tools and want a single source of truth.

📊

At-a-glance menu bar

Ring icons show remaining quota per metric, with monochrome or source-branded colors.

🔌

Four sources built in

Ships with Amp, GitHub Copilot, Codex, and Gemini CLI support out of the box.

📈

Usage history & charts

Track usage trends over time with selectable ranges — Day, Week, Month, or All.

🔮

Forecasting

Projects when you'll run out based on your burn rate. Forecast curves appear on the chart.

🔔

Smart notifications

Get alerted when quota is low, usage spikes, or you're on pace to hit zero before reset.

🔄

Auto-updates

Checks GitHub releases every 6 hours. One-click install & restart from Preferences.

🧩

Extensible by design

Adding a new AI source is a single Swift file. The build script auto-discovers it.

💾

Data management

Export and import usage history as JSON. Delete by source, date range, or entirely.

🩺

Source health monitoring

Tracks fetch failures, surfaces actionable errors, and shows warning indicators.

Supported sources

Track the tools you actually use

Enable whichever ones you use — Rashun won't complain about tools you don't.
